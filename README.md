# HerramientasdeGestion

## Proyecto: GuardIA

GuardIA es un sistema de videovigilancia con inteligencia artificial enfocado en apoyar la detección de eventos relevantes mediante análisis de video.

## Objetivo

Desarrollar un prototipo que permita analizar video mediante inteligencia artificial, organizar la documentación técnica del proyecto y aplicar herramientas de gestión como GitHub para el trabajo colaborativo.

## Tecnologías

- Python
- OpenCV
- YOLO
- Flask o FastAPI
- Railway
- GitHub
